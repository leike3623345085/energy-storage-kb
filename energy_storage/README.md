# 储能行业监控系统

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    储能行业24小时监控系统                      │
├─────────────────────────────────────────────────────────────┤
│  数据采集层  │  数据处理层  │  报告生成层  │  通知推送层      │
├──────────────┼──────────────┼──────────────┼─────────────────┤
│ • 定时搜索   │ • 数据清洗   │ • 日报生成   │ • 消息推送       │
│ • 网站监控   │ • 去重归类   │ • 周报生成   │ • 邮件发送       │
│ • RSS订阅    │ • 关键词提取 │ • 专题报告   │ • 文件存储       │
└──────────────┴──────────────┴──────────────┴─────────────────┘
```

## 定时任务安排

| 时间 | 任务 | 说明 |
|------|------|------|
| 00:00 | 深夜监控 | 美股储能公司动态、欧洲市场 |
| 04:00 | 深夜监控 | 国际储能新闻、行业报告 |
| 08:00 | 早班监控 | 国内储能技术、企业动态 |
| 12:00 | 午班监控 | 午间储能资讯 |
| 16:00 | 下午监控 | 储能政策、市场数据 |
| 18:00 | 日报生成 | 汇总当天资讯，生成日报 |
| 20:00 | 晚班监控 | 晚间储能动态 |
| 周一09:00 | 周报生成 | 汇总本周，生成周报 |

## 报告格式

### 日报结构
1. 概览（资讯数量、热点统计）
2. 热点资讯（分类展示）
3. 趋势分析（关键词热度、行业洞察）
4. 明日关注

### 周报结构
1. 本周大事记
2. 技术趋势分析
3. 重点企业动态
4. 政策解读
5. 下周关注要点

## 文件结构

```
energy_storage/
├── config.py              # 配置文件
├── monitor.py             # 监控脚本
├── generate_report.py     # 报告生成器
├── data/                  # 数据目录
│   ├── news_YYYYMMDD_HHMM.json   # 原始数据
│   └── reports/           # 报告目录
│       ├── report_YYYYMMDD.md    # 日报
│       └── weekly_YYYYMMDD.md    # 周报
└── README.md              # 本文件
```

## 使用说明

### 查看定时任务
```bash
openclaw cron list
```

### 手动触发监控
```bash
python3 energy_storage/monitor.py
```

### 手动生成报告
```bash
python3 energy_storage/generate_report.py
```

### 查看最新报告
```bash
ls -la energy_storage/data/reports/
```

## 扩展建议

1. **添加邮件推送**: 配置SMTP，将报告发送到指定邮箱
2. **接入更多数据源**: 添加RSS订阅、API接口
3. **情感分析**: 对新闻进行情感倾向分析
4. **数据可视化**: 生成趋势图表
5. **异常预警**: 设置关键词预警，重要新闻即时推送
