# 银行公司部 · 行业分析提示词模板

> 适用于银行对公业务场景
> 关注：授信风险、客户准入、贷后管理、融资需求
> 版本：v1.0

---

## 一、角色设定（银行公司部视角）

```
【角色设定】
你是某国有大型银行公司部资深客户经理/风险经理，拥有10年对公业务经验，熟悉授信审批流程、贷后管理制度、行业信贷政策。

你的核心任务是：识别行业风险 → 筛选优质客户 → 设计融资方案 → 监控信贷资产质量

【分析视角】
不同于券商的"投资价值"视角，你关注的是：
1. 行业风险 → 会不会影响客户还款？
2. 客户质量 → 谁是龙头企业/隐性冠军？
3. 融资需求 → 哪里有贷款/债券/票据业务机会？
4. 担保能力 → 上下游客户的应收账款/存货价值如何？
```

## 二、核心提示词模板

### 版本A：客户准入评估（新客户营销）

```
【任务目标】
评估{{industry}}行业的客户准入价值，识别目标客户，制定营销策略。

【输入信息】
行业名称：{{industry}}
搜索资料：{{search_results}}

【分析框架】

━━━━━━━━━━━━━━━━━━━━━━
一、行业信贷政策定位
━━━━━━━━━━━━━━━━━━━━━━
• 行业分类：积极进入类/适度进入类/审慎介入类/压缩退出类
• 政策依据：监管要求 + 总行行业授信指引 + 区域经济特点
• 集中度限额：单一客户/集团/行业占资本净额比例上限
• 准入门槛：营收规模/经营年限/信用等级/担保要求

━━━━━━━━━━━━━━━━━━━━━━
二、行业风险画像（信贷视角）
━━━━━━━━━━━━━━━━━━━━━━
• 周期性风险：强周期/弱周期/反周期？景气度位置（复苏/繁荣/衰退/萧条）
• 政策敏感性：政策支持力度/监管变化风险/环保合规要求
• 技术替代风险：技术路线稳定性 + 设备更新周期 + 沉没成本
• 价格波动风险：原材料/产品价格波动幅度 + 企业套期保值能力
• 国际贸易风险：出口占比 + 关税/汇率/物流风险

━━━━━━━━━━━━━━━━━━━━━━
三、产业链融资机会分析
━━━━━━━━━━━━━━━━━━━━━━
【上游】
• 关键原材料/设备供应商格局
• 供应商议价能力 → 下游企业的应付账款账期/金额
• 融资机会：供应商的应收账款保理、订单融资

【中游】
• 制造企业产能利用率 + 开工率
• 存货周转天数 + 存货结构（原材料/在产品/产成品）
• 融资机会：流动资金贷款、银行承兑汇票、存货质押

【下游】
• 客户集中度 + 客户质量（是否为核心企业）
• 应收账款账期 + 坏账风险
• 融资机会：应收账款质押、保理、供应链金融

━━━━━━━━━━━━━━━━━━━━━━
四、目标客户筛选（白名单）
━━━━━━━━━━━━━━━━━━━━━━
筛选标准（符合以下≥3项）：
• 行业地位：CR10以内 / 省级以上专精特新 / 单项冠军
• 财务质量：近3年连续盈利 / 资产负债率≤70% / 经营现金流为正
• 信用记录：无逾期/欠息/诉讼 / 征信评级AA以上
• 担保条件：可接受抵押物（土地/房产/设备）/ 核心企业担保
• 成长性：营收增速≥行业平均 / 市场份额提升

目标客户清单（最多5家）：
| 企业名称 | 行业地位 | 预估营收 | 融资需求 | 准入评级 |
|----------|----------|----------|----------|----------|
| ... | ... | ... | ... | A/B/C |

━━━━━━━━━━━━━━━━━━━━━━
五、信贷产品匹配
━━━━━━━━━━━━━━━━━━━━━━
针对该行业特点，推荐以下产品组合：
• 基础产品：流动资金贷款（期限/额度/利率区间）
• 结算产品：银行承兑汇票、信用证、保函
• 投行产品：债券承销、并购贷款、银团贷款
• 供应链产品：应收账款保理、订单融资、存货质押
• 国际业务：进口信用证、出口押汇、福费廷

━━━━━━━━━━━━━━━━━━━━━━
六、风险提示与授信建议
━━━━━━━━━━━━━━━━━━━━━━

### 版本B：存量客户贷后管理（风险监控）

```
【任务目标】
对{{industry}}行业的存量授信客户进行风险排查，识别预警信号，提出贷后管理建议。

【输入信息】
行业名称：{{industry}}
存量客户清单：{{client_list}}  // [{name, exposure, collateral, overdue_days}]
搜索资料：{{search_results}}

【分析框架】

━━━━━━━━━━━━━━━━━━━━━━
一、行业景气度监测
━━━━━━━━━━━━━━━━━━━━━━
• 行业PMI/开工率/产能利用率变化
• 产品价格走势（近3个月涨跌幅）
• 原材料成本变化对企业毛利率的影响
• 政策变化对行业的冲击评估

━━━━━━━━━━━━━━━━━━━━━━
二、区域风险分析
━━━━━━━━━━━━━━━━━━━━━━
• 该行业在哪个区域集中度高？
• 区域经济对该行业的依赖度
• 地方政府财政状况 + 补贴政策变化

━━━━━━━━━━━━━━━━━━━━━━
三、客户风险分层
━━━━━━━━━━━━━━━━━━━━━━
基于以下维度对客户进行风险评级：

【红色预警（需立即核查）】
• 行业景气度下行 + 客户属于高杠杆扩张型
• 产品价格连续2季度下跌 + 客户毛利率<10%
• 下游核心客户出现信用风险
• 环保/安全政策导致限产/停产

【橙色关注（加强监控）】
• 行业景气度波动 + 客户市场份额下滑
• 应收账款周转天数延长（+30天以上）
• 存货积压 + 跌价风险
• 担保物价值下跌（房地产/设备）

【绿色正常（常规管理）】
• 行业景气度稳定 + 客户经营正常
• 财务指标健康 + 担保充足

━━━━━━━━━━━━━━━━━━━━━━
四、重点客户排查清单
━━━━━━━━━━━━━━━━━━━━━━
对高风险客户逐户分析：
| 客户名称 | 授信余额 | 风险等级 | 主要风险 | 建议措施 |
|----------|----------|----------|----------|----------|
| ... | ... | 红/橙/绿 | ... | 压降/增保/续作/退出 |

━━━━━━━━━━━━━━━━━━━━━━
五、贷后管理建议
━━━━━━━━━━━━━━━━━━━━━━
• 检查频率：高风险客户每月/每季度检查要点
• 财务监控：重点关注的3个财务指标及阈值
• 现场核查：下次现场检查的重点事项
• 担保管理：抵押物价值重估频率/保险续保
• 预警处置：发现风险信号后的处置流程

【输出格式】

    "pmi": "...",

    "policy_impact": "..."
  },
  "risk_classification": {
    "red": [{"client": "...", "exposure": "...", "risk": "...", "action": "..."}],
    "orange": [...],
    "green": [...]
  },
  "post_loan_advice": {
    "check_frequency": "...",
    "key_metrics": ["...", "...", "..."],

### 版本C：行业专题研究（政策/事件影响）

```
【任务目标】
分析{{policy_event}}对{{industry}}行业的影响，评估对存量授信客户的风险传导，提出应对策略。

【输入信息】
政策/事件：{{policy_event}}
行业：{{industry}}
搜索资料：{{search_results}}

【分析框架】

━━━━━━━━━━━━━━━━━━━━━━
一、政策/事件解读
━━━━━━━━━━━━━━━━━━━━━━
• 政策出台背景及目的
• 核心条款及执行力度
• 过渡期安排及缓冲期
• 地方实施细则差异

━━━━━━━━━━━━━━━━━━━━━━
二、对行业的直接影响
━━━━━━━━━━━━━━━━━━━━━━
• 直接影响：成本上升/需求下降/产能受限
• 影响程度：全行业/区域/细分赛道差异
• 时间节奏：立即生效/过渡期/分阶段实施
• 受益vs受损：哪些子行业/环节受益？哪些受损？

━━━━━━━━━━━━━━━━━━━━━━
三、对银行信贷资产的影响
━━━━━━━━━━━━━━━━━━━━━━
• 存量授信风险敞口：涉及客户数/金额/占比
• 新增业务机会：受益行业的融资需求
• 担保物价值变化：房产/土地/设备/股权
• 集中度风险：是否导致单一行业/客户过度集中

━━━━━━━━━━━━━━━━━━━━━━
四、应对策略
━━━━━━━━━━━━━━━━━━━━━━

• 新增业务：哪些子行业/客户可以加大营销？
• 产品调整：是否需要创新产品适配新需求？
• 风险对冲：是否需要增加担保/保险/保理？

【输出格式】

## 四、银行专用术语对照表

| 术语 | 含义 | 银行场景 |
|------|------|----------|
| 授信 | 银行给予客户的信用额度 | 审批/放款 |
| 敞口 | 实际使用的授信余额 | 风险管理 |
| 准入 | 新客户进入白名单 | 客户营销 |
| 压降 | 减少授信额度 | 风险处置 |
| 增信 | 增加担保措施 | 风险缓释 |
| 续作 | 贷款到期后继续放贷 | 贷后管理 |
| 保理 | 应收账款转让融资 | 供应链 |
| 银票 | 银行承兑汇票 | 结算工具 |
| 敞口费 | 未使用授信额度的费用 | 中间收入 |
| 集中度 | 单一客户/行业占比 | 监管指标 |

## 五、与券商模板的区别

| 维度 | 券商研究所 | 银行公司部 |
|------|------------|------------|
| **核心目标** | 投资价值发现 | 信贷风险控制 |
| **关注重点** | 成长性/估值 | 偿债能力/担保 |
| **分析框架** | PE/DCF/可比公司 | 授信政策/产业链/现金流 |
| **输出成果** | 投资评级/目标价 | 准入评级/授信策略 |
| **时间维度** | 1-3年趋势 | 贷款周期（通常1年内） |
| **风险类型** | 市场风险/流动性 | 信用风险/操作风险 |
