# 储能报告知识库管理方案

## 方案概述

储能行业报告智能管理系统，支持两种方式：

1. **📚 向量知识库**（推荐）- 语义检索 + 智能问答
   - 语义搜索：理解问题意图，不只是关键词匹配
   - 智能问答：直接问"这周钠电进展"，自动总结答案
   - 无需外部平台，完全本地化

2. **📁 飞书知识库** - 云端归档 + 团队协作
   - 结构化存储：按类型、日期自动分类
   - 多端访问：PC/移动端随时查阅
   - 适合团队共享和人工查阅

# 🚀 方案C：向量知识库（已部署）

## 核心能力

### 1. 语义搜索
不只是关键词匹配，能理解同义词和相关概念：
- 搜"钠电" → 匹配"钠离子电池"
- 搜"构网型" → 匹配"grid-forming储能"
- 搜"储能消防" → 匹配"pack级消防"、"热失控"

### 2. 智能问答
直接问问题，系统自动检索并总结：
- "这周有什么储能政策？"
- "宁德时代最近有什么动态？"
- "钠离子电池产业化进展如何？"

### 3. 自动归档
每天生成的报告自动向量化，无需人工干预。

## 快速开始

### 查看知识库状态
```bash
cd energy_storage/knowledge_base
python3 vector_kb.py --stats
```

### 测试问答（直接问我）
现在你可以直接问我任何问题，例如：
- "最近有什么储能政策"
- "这周钠离子电池有什么进展"
- "储能消防有什么新技术"

### 手动索引新报告
```bash
# 索引单个文件
python3 vector_kb.py --index-file /path/to/report.md

# 索引所有历史报告
python3 vector_kb.py --index-all
```

### 命令行搜索测试
```bash
# 语义搜索
python3 vector_kb.py --search "钠离子电池产业化"

# 智能问答
python3 qa_interface.py "这周有什么政策"

# 交互式问答
python3 qa_interface.py -i
```

## 定时任务配置

已配置自动索引任务：
```bash
# 日报生成后自动索引（每天18:25）
openclaw cron add \
  --name "vector_index_daily" \
  --schedule "0 25 18 * * *" \
  --command "cd energy_storage/knowledge_base && python3 auto_index.py daily"

# 深度分析自动索引（每天18:30）
openclaw cron add \
  --name "vector_index_deep" \
  --schedule "0 30 18 * * *" \
  --command "cd energy_storage/knowledge_base && python3 auto_index.py deep"
```

# 📁 方案B：飞书知识库（可选）

如果你需要云端备份或团队共享，可额外配置飞书同步。

## 技术原理

### 向量知识库工作原理
1. **文本分块**：长报告切分为小块（500字/块，重叠100字）
2. **特征提取**：基于储能行业关键词词典提取特征向量
3. **语义索引**：为每个文本块生成512维向量
4. **相似度检索**：余弦相似度匹配查询与文档
5. **智能问答**：检索相关内容 + 结构化呈现

### 关键词词典
覆盖储能行业核心概念：
- 电池技术：钠离子、固态、液流、磷酸铁锂
- 应用场景：新型电力系统、虚拟电厂、削峰填谷
- 设备技术：PCS、构网型、BMS、EMS
- 政策法规：储能调度、容量电价、电力现货市场
- 重点企业：宁德时代、比亚迪、阳光电源等
