## 系统概述
24小时持续监控储能行业最新动态，自动生成日报/周报并邮件推送。

## 核心定位
> **新型电力系统的"稳定器"** —— 从根本上解决风电/光伏的"间歇性"，实现削峰填谷

### 战略定位
- **根本使命**：解决风光间歇性，实现削峰填谷
- **战略角色**：新型电力系统的"稳定器"、新能源的刚需配套
- **支撑目标**：服务"双碳"战略

### 技术覆盖领域
| 领域 | 关键技术/方向 |
|------|---------------|
| **储能系统集成** | 压缩空气储能、飞轮储能、储热/储冷、氢能储能 |
| **电池多元化** | 锂电池(LFP)、钠离子电池、液流电池(全钒/锌溴/铁铬)、固态/半固态电池 |
| **PCS与电力电子** | 储能变流器、构网型储能(grid-forming)、高压直挂/级联式储能 |
| **消防与安全** | pack级消防、电池热失控防控、全氟己酮等气体消防 |
| **智能运维** | BMS/EMS、数字孪生、储能云平台、远程监控 |
| **虚拟电厂** | VPP聚合、源网荷储一体化、车网互动(V2G)、微电网 |

### 政策关注重点
- **标准统一**：储能并网标准、消防安全标准
- **调度机制**：储能调度规则、电力现货市场准入
- **商业模式**：容量租赁、辅助服务收益、电力交易

### 日报/周报聚焦方向
1. 新型电力系统建设进展
2. 风光大基地配套储能项目
3. 储能系统集成技术突破（压缩空气/飞轮/氢能等）
4. 电池多元化产业化动态（钠电/液流/固态）
5. 构网型储能技术进展
6. 储能消防与安全标准
7. 虚拟电厂商业模式与政策
8. 电力市场化改革对储能的影响
