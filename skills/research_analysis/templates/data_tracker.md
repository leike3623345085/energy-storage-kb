# 数据追踪模板

## 使用说明

本模板用于追踪关键指标的长期趋势，建立看板式的数据监控。

## 追踪指标体系

### 1. 市场规模指标

| 指标名称 | 单位 | 数据来源 | 更新频率 |
|---------|------|---------|---------|
| 新型储能累计装机 | GW/GWh | 国家能源局 | 月度 |
| 储能系统招标量 | GWh | 公开招投标 | 周度 |
| 储能项目备案量 | GW | 各省发改委 | 月度 |
| 储能投资额 | 亿元 | 公开数据 | 季度 |

### 2. 价格指标

| 指标名称 | 单位 | 数据来源 | 更新频率 |
|---------|------|---------|---------|
| 储能系统中标均价 | 元/Wh | 中标公告 | 周度 |
| 电芯价格 | 元/Wh | 行业报价 | 周度 |
| 碳酸锂价格 | 万元/吨 | 上海有色网 | 日度 |
| PCS均价 | 元/W | 中标公告 | 月度 |

### 3. 政策指标

| 指标名称 | 单位 | 数据来源 | 更新频率 |
|---------|------|---------|---------|
| 国家级政策发布 | 项 | 部委网站 | 实时 |
| 地方政策发布 | 项 | 各省网站 | 实时 |
| 标准发布 | 项 | 标准化委员会 | 月度 |

### 4. 技术路线指标

| 技术路线 | 追踪维度 | 数据来源 |
|---------|---------|---------|
| 锂离子电池 | 装机占比、项目数量 | 公开数据 |
| 钠离子电池 | 项目进展、产业化动态 | 公开数据 |
| 液流电池 | 项目进展、成本趋势 | 公开数据 |
| 压缩空气储能 | 项目进展、技术突破 | 公开数据 |
| 固态电池 | 研发进展、量产时间 | 公开数据 |
| 构网型储能 | 项目数量、技术成熟度 | 公开数据 |

## 数据存储

建议将追踪数据存储在结构化文件中：

```json
{
  "tracker_version": "1.0",
  "industry": "储能",
  "last_updated": "2026-03-22",
  "metrics": {
    "installed_capacity": {
      "unit": "GW",
      "frequency": "monthly",
      "data": [
        {"date": "2026-03", "value": 136, "growth_yoy": 84},
        {"date": "2026-02", "value": 128, "growth_yoy": 78}
      ]
    },
    "system_price": {
      "unit": "元/Wh",
      "frequency": "weekly",
      "data": [
        {"date": "2026-W12", "value": 0.45, "change_wow": -2},
        {"date": "2026-W11", "value": 0.46, "change_wow": -1}
      ]
    }
  }
}
```

## 使用方法

```python
from skills.research_analysis.templates.data_tracker import DataTracker

tracker = DataTracker(industry="储能")

# 添加新数据
tracker.update_metric("installed_capacity", date="2026-03", value=136)

# 生成看板
report = tracker.generate_dashboard()

# 异常检测
alerts = tracker.check_anomalies()
```
