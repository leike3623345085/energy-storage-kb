## Mock 工具

创建了完整的 ExcelJS Mock 工具 (`tests/mocks/exceljs.mock.js`)，包括：

- `createMockCell()` - 创建 Mock 单元格
- `createMockRow()` - 创建 Mock 行
- `createMockWorksheet()` - 创建 Mock 工作表
- `TestTemplates` - 预定义的测试数据模板
  - `salesData()` - 标准销售数据表
  - `complexWithBlanks()` - 带空白行的复杂表
  - `headerOnly()` - 只有表头
  - `emptySheet()` - 空工作表
  - `mostlyNumbers()` - 大量数字
  - `withFormulas()` - 带公式
  - `withDates()` - 带日期
  - `withRichText()` - 富文本对象
  - `chineseKeywords()` - 中文关键词
  - `multipleHeaderCandidates()` - 多行表头候选

## 运行说明

```bash

npm install
```

